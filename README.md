# CodeAlpha_MatrixOperations

This repository contains **TASK 2: Medium - Matrix Operations** developed during my **C Programming Internship** at **CodeAlpha**.

## 📌 Project Overview
The goal of this project is to implement a modular and menu-driven C program to perform fundamental operations on multi-dimensional arrays (matrices). 

To ensure clean and readable code structure, the project is broken down into separate, dedicated **functions** for input, output, and execution of each core matrix operation.

## 🚀 Features
- **Matrix Addition:** Adds two matrices of the same dimensions.
- **Matrix Multiplication:** Multiplies two matrices after validating the condition that the number of columns in Matrix A must equal the number of rows in Matrix B.
- **Matrix Transpose:** Interchanges rows and columns of a user-defined matrix.
- **Modular Code Design:** Uses separate C functions for every operation to maximize reusability.
- **Formatted Grid Display:** Prints matrices in a clean row-and-column layout using tab spaces.

## 🛠️ Tech Stack & Concepts Used
- **Language:** C
- **Core Concepts:** 2D Arrays (Multi-dimensional Arrays), Functions (Passing arrays to functions), Loops (`nested for loops` for matrix traversal), and Decision-Making structures.
